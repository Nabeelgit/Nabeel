3D Shooter Game

Social Network

Social Network is a recreation of Facebook from 2008, I used PHP and MongoDB to create this project. It has a login and signup system, a messaging, friend, profile, posts, likes, comments, and search system.

Windows XP clone

Windows XP clone is a project made by me using HTML, CSS, and JavaScript, it's a simple project that involves creating a clone of the windows xp operating system. You can open apps, resize, maximize, and minimize them. There's also a start menu which you can perform various tasks in.

Enginon Search

Enginon is a search engine which I worked on for over a year. It's made in PHP and MongoDB (initally MySQL). I also made the crawler myself to crawl the web and index the web pages. It has a translation, unit conversion, weather, etc.
